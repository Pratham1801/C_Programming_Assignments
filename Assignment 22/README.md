# Assignment 22
 This folder contains problems related to digits in which input is taken from user as a Integer, problems are:- 
 1. Used to accept N number and find even numbers from them
 2. Used to accept N number and find frequency difference between even and odd numbers
 3. Used to accept N number and find 11 is present or not
 4. Used to accept N number and find 11's number of count in data
 5. Used to accept N number and find iNo's number of count in data
 
 Commands used to execute these codes are:- 
 
 gcc filename.c -o myexe 
 
 myexe
