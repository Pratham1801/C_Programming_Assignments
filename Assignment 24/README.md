# Assignment 24
 This folder contains problems related to digits in which input is taken from user as a Integer, problems are:- 
 1. Used to accept N number and find maximum number among them
 2. Used to accept N number and find minimum number among them
 3. Used to accept N number and find difference between maximum and minimum number
 4. Used to accept N number and find numbers having 3 digits in it
 5. Used to accept N number and and returning sum of each number
 
 Commands used to execute these codes are:- 
 
 gcc filename.c -o myexe 
 
 myexe
