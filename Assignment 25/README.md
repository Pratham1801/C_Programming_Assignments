# Assignment 25
 This folder contains problems related to digits in which input is taken from user as a Integer, problems are:- 
 1. Used to accept N number and return difference between summation of even and odd
 2. Used to accept N number and return number divisible by 5
 3. Used to accept N number and return number even and divisible by 5
 4. Used to accept N number and return number divisible by 5 and 3
 5. Used to accept N number and return number divisible by 11
 
 Commands used to execute these codes are:- 
 
 gcc filename.c -o myexe 
 
 myexe
