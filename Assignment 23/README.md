# Assignment 23
 This folder contains problems related to digits in which input is taken from user as a Integer, problems are:- 
 1. Used to accept N number and find NO is present or not
 2. Used to accept N numbers and No and display No's index number occured first time
 3. Used to accept N numbers and No and display No's index number occured last time
 4. Used to accept N numbers and a range and display the numbers int that range from data
 5. Used to accept N numbers from user and print the product of all odd number from data
 
 Commands used to execute these codes are:- 
 
 gcc filename.c -o myexe 
 
 myexe
